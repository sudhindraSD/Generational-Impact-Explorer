About This Project

This project demonstrates how SQL databases can be used to analyze and compare generational trends in society.It combines a user-friendly frontend with a powerful backend to deliver meaningful insights.

The data used in this project includes key metrics such as average income, education levels, technology adoption scores, social media usage, and environmental awareness. These metrics provide a comprehensive view of how different generations have shaped and been shaped by societal changes.

The project is built using modern web technologies, including HTML, CSS, JavaScript, Node.js, and MySQL. It serves as a practical example of how data can be visualized and compared to drive better understanding and decision-making.
